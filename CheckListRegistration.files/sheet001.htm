Проект Evroopt
Название (Аббр.): Модуль регистрации (Рег)
Описание: Модуль регистрации содержащий в себе политику конфиденциальности, без подтверждения которой дальнейшая регистрация невозможна. Форма регистрации содрежащее несколько обязательных полей(Имя, фамилия, мобильный телефон, пароль, подтвердить пароль) и одно необязательное(email)
Статус: Согласован
Дата 24.12.2023
Окружение Lenovo Legion y540-15irh
ОС Windows 11 Pro Версии 23H2
Microsoft Edge Версия 120.0.2210.77 (64 бит)
